3d-model-to-pdf-conversion

Autodesk 3D Files(DWG & DXF) to PDF Conversion and Automation using free Autodesk DWG True Viewer 2016 and Autolisp Scripts

View on GitHub Download .zip Download .tar.gz

Welcome to 3d-model-to-pdf-conversion Project Page.

This small project allows developers to use Autodesk's freely available DWG/DXF model viewer and generate 2D pdf files from them.This project automates DWG/DXF to pdf conversion using command line tool(accoreconsole.exe).
